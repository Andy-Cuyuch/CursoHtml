Etiquetas semanticas estructurales

header -- Representa la cabezera de un sitio o de una seccion
main -- Define la seccion principal del documento (solo existe una etiqueta por documento)
footer -- Representa un pie de pagina de una sitio web
nav -- Representa una navegacion
article -- Representa una seccion de auto contenido
aside -- representa un contenido secundario (Sonas de publicidad)
section -- Representan una seccion de contenido generico
address -- Representa una dirección fisica
